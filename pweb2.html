Sejarah HTML

Bahasa Pemrograman HTML (Hypertext Markup Language) terus mengalami perkembangan sejak pertama kali dikenalkan. Pada Desember 1997, World Wide Web Consortium (W3C) memperkenalkan HTML 4.0. Versi ini membawa banyak fitur baru, salah satunya adalah dukungan untuk CSS (Cascading Style Sheets), yang memungkinkan pembuatan aplikasi web menjadi lebih dinamis dan terstruktur.

Dua tahun kemudian, pada tahun 1999, muncul HTML 4.1. Setelah itu, lahirlah XHTML 1.0. Walaupun secara prinsip spesifikasi XHTML sama seperti HTML 4.01, namun XHTML menerapkan aturan penulisan yang lebih ketat berdasarkan standar XML (eXtended Markup Language). Salah satu perbedaan utamanya adalah, dalam XHTML, semua tag dan atribut wajib ditulis dengan huruf kecil. Sedangkan pada HTML, tag dan atribut boleh menggunakan huruf besar, kecil, atau kombinasi keduanya. Setelah XHTML 1.0, dikembangkan pula XHTML 2.0, namun versi ini kurang mendapat sambutan dari para pengembang dan vendor.

Sementara itu, perkembangan dunia web terus berlanjut. Muncullah HTML5, yang dikembangkan oleh organisasi WHATWG (Web Hypertext Application Technology Working Group), bukan lagi hanya oleh W3C. HTML5 hingga kini belum sepenuhnya final, tetapi sudah banyak digunakan dan diyakini akan menjadi standar web masa depan. Bahkan, meski belum resmi final, HTML5 telah menarik perhatian karena mendukung banyak fitur modern seperti pemutaran video langsung di browser tanpa plugin tambahan.

Beberapa browser sudah mulai mendukung HTML5, di antaranya:

  • Chrome sejak versi 1.0
  • Firefox sejak versi 1.5
  • Internet Explorer mulai versi 8
  • Opera sejak versi 9.0
  • Safari sejak versi 1.3

    • Dalam HTML5, aturan penulisan menjadi lebih bebas. Tidak ada keharusan menggunakan huruf kecil untuk tag atau atribut, dan bahkan, nilai atribut boleh ditulis dengan atau tanpa tanda petik. Browser modern sudah bisa memahami berbagai variasi penulisan tersebut tanpa masalah. Dengan perkembangan HTML dari versi 4.0, XHTML, hingga HTML5, dunia web menjadi semakin kaya, dinamis, dan mendukung berbagai kebutuhan modern pengguna internet.

    Struktur Dasar HTML

    Setiap dokumen HTML memiliki struktur dasar atau susunan file sebagai berikut :

                        <html>
                        <head>
                            <title>Judul Halaman</title>
                        </head>
                        <body>
                            Berisi tentang text, gambar,
                            atau apapun yang
                            tampil pada dokumen web.
                        </body>
                    </html>

    Dalam pembuatan halaman web menggunakan HTML, struktur file selalu diawali dengan tag dan diakhiri dengan tag . Semua isi halaman web akan berada di dalam tag ini.

    Di dalam tag , terdapat dua bagian utama:

    1. Bagian <head>...</head> Bagian ini disebut header dari halaman HTML. Isi dari tag <head> tidak akan ditampilkan di halaman web browser. Tag <title>...</title>, yang berfungsi untuk menampilkan judul halaman pada title bar (bagian atas jendela browser). Selain <title>, di dalam <head> juga bisa berisi pengaturan lain seperti link ke file CSS, meta description, dan sebagainya. Bagian ini biasanya digunakan untuk menuliskan informasi seperti:
    2. Bagian <body>...</body> Bagian ini adalah tempat semua konten yang ditampilkan ke pengguna di halaman web browser. Dalam tag <body>, kita bisa menuliskan berbagai jenis informasi, seperti: eks dengan berbagai format,Gambar, Tautan (link), Video, Dan elemen-elemen lainnya. Dengan kata lain, bagian <head> berisi informasi tentang halaman, sedangkan bagian <body> berisi isi atau tampilan halaman yang dilihat pengguna.

    HTML 5

    Sebuah dokumen HTML5 seperti berikut ini, cukup ditulis dengan Text Editor ataupun HTML Editor : 

                                <!DOCTYPE HTML>
                                <html>
                                    <head>
                                        <title>Tes HTML5</title>
                                    </head>
                                <body>
                                    Selamat belajar HTML5
                                </body>
                                </html>

    Berikut ini penjelasan dari dokumen HTML5 di atas :

    Sebuah dokumen HTML5 diawali dengan <!DOCTYPE HTML>.

    • Tanda seperti <html> disebut tag. Sebuah tag menyatakan sebuah elemen dalam dokumen HTML.
    • Beberapa tag berpasangan, seperti <head> dengan </head>.
    • Tidak semua tag berpasangan. Contohnya, <br> tidak memiliki pasangan. Pada XHTML, tag seperti ini diharuskan menggunakan tanda / sebelum > (misal: <br />).
    • Pasangan <html>...</html> menyatakan awal dan akhir dokumen HTML.
    • Di dalam <html>...</html> terdapat pasangan <head>...</head> dan <body>...</body>.
    • Pasangan <head>...</head> berisi bagian judul dokumen HTML, minimal harus ada pasangan <title>...</title>.
    • Pasangan <body>...</body> berisi bagian tubuh dokumen. Pada contoh, teks "Selamat belajar HTML5" muncul pada tubuh dokumen.

    Berikut ini adalah elemen-elemen HTML 4.01 yang tidak lagi digunakan di HTML5:

    • <acronym>
    • <applet>
    • <basefont>
    • <big>
    • <center>
    • <dir>
    • <font>
    • <frame>
    • <frameset>
    • <noframes>
    • <strike>
    • <tt>

    Kode Warna

    Dalam pengaturan warna menggunakan kode RGB yang ditampilkan dalam nilai heksadesimal. Setiap dua digit kode menunjukkan intensitas kombinasi warna merah, hijau, dan biru.

    Contoh kode warna:

    Color Hexadecimal
    White #FFFFFF
    Red #FF0000
    Blue #0000FF
    Black #000000
    Green #00FF00
    Magenta #FF00FF
    Cyan #00FFFF
    Aquamarine #70DB93
    Violet #9F5F9F
    Copper #B87333
    Yellow #FFFF00
    Chocolate #5C3317
    Brass #B5A642
    Pink #FF6EC7

    Atribut Elemen <body>

    Berikut atribut-atribut yang bisa digunakan pada elemen <body>:

    • BACKGROUND = Lokasi dan nama file gambar latar belakang dokumen
    • BGCOLOR = Warna latar belakang dokumen (default: putih)
    • TEXT = Warna teks dokumen (default: hitam)
    • LINK = Warna link (default: biru)
    • VLINK = Warna visited link (default: ungu)
    • ALINK = Warna aktif link (default: merah)
